1. 新建Hello.java文件，编写如下代码

```java
/**
 * @author xuwei
 * @since 1.8
 * @version 1
 */
public class Hello {
    public static void main(String[] args) {
        System.out.print("hello world");
    }
}
```

2. 执行javadoc生成文档

```java
javadoc -encoding UTF-8 -charset UTF-8 Hello.java
```

然后就会生成一大堆文件，在浏览器中打开index.html即可。
